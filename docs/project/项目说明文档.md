## 1. 项目概述

### 1.1 项目名称

黑科易购 - 黑龙江科技大学特色校园服务平台

### 1.2 项目定位

黑科易购是专为黑龙江科技大学师生打造的一站式校园服务平台，致力于提供便捷、高效、安全的校园生活服务，包括校园外卖、快递服务、校园信息发布、学生社区等功能，旨在提升校园生活品质和信息化水平。

### 1.3 核心价值

- **便捷性**：整合多项校园服务，实现一站式访问
- **高效性**：优化服务流程，提升用户体验
- **安全性**：严格的用户认证和数据保护机制
- **校园特色**：针对黑龙江科技大学校园环境和师生需求定制开发

## 2. 系统架构

### 2.1 整体架构

- **前端层**：Web端、小程序端、管理后台
- **API网关层**：统一接口管理、认证授权、限流熔断
- **应用服务层**：微服务架构，包含10个核心服务模块
- **数据层**：MySQL主从、Redis缓存、Elasticsearch搜索
- **基础设施层**：容器化部署、服务发现、配置中心

### 2.2 核心服务模块

1. **服务网关 (api-gateway)**：统一入口，路由转发，权限控制
2. **用户服务 (service-user)**：用户管理、认证授权、个人信息
3. **商品服务 (service-product)**：商品管理、分类、搜索、推荐
4. **订单服务 (service-order)**：订单管理、状态流转、订单统计
5. **支付服务 (service-pay)**：支付集成、交易记录、退款处理
6. **营销服务 (service-marketing)**：优惠券、促销活动、积分管理
7. **外卖服务 (service-delivery)**：食堂外卖、商家管理、配送服务
8. **配送服务 (service-logistics)**：配送员管理、路线规划、位置追踪
9. **校园服务 (service-campus)**：校园地图、公告活动、学业辅助
10. **社区服务 (service-community)**：学生社区、闲置交易、互助平台

## 4. 技术栈

### 4.1 后端技术栈

- **开发语言**：Java 17
- **框架**：Spring Boot 3.x, Spring Cloud 2022.x, Spring Security
- **数据库**：MySQL 8.3.x, Redis 7.2.x, Elasticsearch 8.x
- **消息队列**：RocketMQ 5.x
- **服务治理**：Nacos, Sentinel, Seata
- **API文档**：SpringDoc OpenAPI 2.0

### 4.2 前端技术栈

- **Web端**：Vue 3.5, Vite 5.x, TypeScript 5.x, Element Plus
- **小程序端**：Taro 3.6+, Vue 3.x, Pinia, TypeScript
- **构建工具**：Webpack 5.x, Vite
- **UI组件库**：Element Plus, NutUI

### 4.3 运维技术栈

- **容器化**：Docker, Kubernetes
- **CI/CD**：Jenkins, GitLab CI
- **监控告警**：Prometheus, Grafana, ELK
- **日志管理**：Logback, ELK Stack

## 5. 项目特色

### 5.1 黑龙江科技大学特色功能

- **校园地图导航**：精准定位校园建筑物，支持导航功能
- **食堂外卖服务**：整合校内食堂资源，提供便捷点餐服务
- **学业辅助工具**：课程表、考试安排查询等学习支持
- **校园社区**：学生互助、闲置交易等社交功能

### 5.2 性能优化

- **分布式缓存**：Redis集群加速数据访问
- **读写分离**：MySQL主从架构提升并发能力
- **异步处理**：消息队列处理耗时操作
- **CDN加速**：静态资源分发优化

### 5.3 安全保障

- **身份认证**：JWT Token认证机制
- **权限控制**：细粒度的RBAC权限模型
- **数据加密**：敏感信息加密存储
- **防SQL注入**：参数化查询，ORM框架防护
- **防XSS攻击**：输入验证，输出转义
