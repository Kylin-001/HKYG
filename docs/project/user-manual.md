## 1. 平台概述

黑科易购是专为**黑龙江科技大学**师生打造的一站式校园服务平台，提供以下核心服务：

| 服务 | 说明 | 访问方式 |
|------|------|----------|
| 🛒 校园商城 | 商品购买、购物车、订单管理 | Web端 / 小程序 |
| 🍔 外卖点餐 | 商家浏览、菜品下单、配送跟踪 | Web端 / 小程序 |
| 🔄 二手市场 | 发布/浏览二手商品、在线沟通 | Web端 / 小程序 |
| 🔍 失物招领 | 发布/查找失物、寻物启事 | Web端 / 小程序 |
| 💬 社区论坛 | 发帖回帖、互动交流 | Web端 |
| 🏫 校园服务 | 课表、成绩、图书馆、地图 | Web端 |

## 2. 快速开始

### 2.1 访问平台

```
Web端: http://localhost:5173 (开发环境)
小程序: 微信搜索"黑科易购"
```

### 2.2 注册账号

1. 点击页面右上角 **"登录/注册"**
2. 选择 **"注册新账号"**
3. 输入手机号，获取验证码
4. 设置登录密码（8-20位，含字母和数字）
5. 完成注册，自动登录

### 2.3 登录方式

| 方式 | 说明 |
|------|------|
| 密码登录 | 手机号 + 密码 |
| 短信登录 | 手机号 + 验证码（无需密码） |
| 微信登录 | 扫码授权（小程序端） |

## 3. 功能使用指南

### 3.1 🛒 校园商城

#### 浏览商品
1. 点击导航栏 **"商品"** 或首页 **"精选商品"**
2. 使用顶部 **搜索框** 搜索关键词
3. 通过左侧 **分类筛选** 缩小范围
4. 支持 **排序**: 价格↑↓ / 销量 / 新品 / 评分

#### 购买商品
1. 进入商品详情页
2. 选择规格/数量
3. 点击 **"加入购物车"** 或 **"立即购买"**
4. 确认收货地址 → 选择支付方式 → 提交订单

#### 购物车管理
- 点击右上角 **🛒 图标** 进入购物车
- 可修改数量、删除商品、全选结算
- 优惠券在结算页自动匹配可用优惠

#### 订单管理
- 进入 **"个人中心" → "我的订单"**
- 订单状态: 待支付 → 待发货 → 待收货 → 已完成
- 支持: 确认收货、申请退款、评价、再次购买

### 3.2 🍔 外卖点餐

#### 下单流程
1. 点击 **"外卖"** 导航
2. 浏览商家列表（可按距离/评分/销量排序）
3. 进入商家 → 选择菜品（可选规格/口味）
4. 加入外卖购物车 → 结算
5. 填写配送地址和时间 → 支付

#### 配送跟踪
- 订单详情页实时显示配送状态
- 支持 WebSocket 推送，无需刷新
- 配送状态: 商家接单 → 制作中 → 取餐中 → 配送中 → 已送达

### 3.3 🔄 二手市场

#### 发布二手商品
1. 点击 **"二手" → "发布商品"**
2. 填写: 标题、描述、价格、成色、分类
3. 上传商品图片（最多9张）
4. 选择交易方式: 面交 / 校内配送 / 邮寄
5. 发布后可在 **"我的发布"** 中管理

#### 购买/交换二手商品
- 浏览二手列表，筛选: 分类/成色/价格区间
- 点击进入详情，联系卖家（站内消息）
- 协商一致后确认交易

### 3.4 🔍 失物招领

#### 发布失物/寻物
1. 进入 **"社区 → 失物招领"**
2. 点击 **"发布"** 按钮
3. 选择类型: **我丢了东西** / **我捡到东西**
4. 填写物品名称、特征描述、丢失/捡到地点
5. 上传图片（如有），留下联系方式
6. 发布后等待其他用户联系

#### 查找失物
- 使用搜索框输入关键词
- 按 **类型** (失物/寻物)、**地点**、**时间** 筛选
- 找到匹配信息后通过站内信联系发布者

### 3.5 💬 社区论坛

#### 发帖
1. 进入 **"社区"** 选择对应板块
2. 点击 **"发帖"** 按钮
3. 填写标题和内容（支持富文本和图片）
4. 选择所属板块和标签
5. 发布

#### 互动
- **点赞/踩**: 表达对内容的看法
- **收藏**: 保存感兴趣的帖子
- **评论**: 参与讨论
- **分享**: 分享到其他平台

### 3.6 🏫 校园服务

| 服务 | 入口 | 功能说明 |
|------|------|----------|
| 课表查询 | 校园服务 → 课表 | 查看本周课程安排 |
| 成绩查询 | 校园服务 → 成绩 | 查看学期成绩（需绑定教务系统） |
| 图书馆 | 校园服务 → 图书馆 | 图书搜索/预约/借阅记录 |
| 校园地图 | 校园服务 → 地图 | 建筑/设施/路线导航 |
| 宿舍报修 | 校园服务 → 报修 | 提交维修工单，跟踪进度 |

## 4. 个人中心

### 4.1 基本信息
- **头像/昵称**: 点击右上角头像 → 个人资料
- **账户设置**: 修改密码、绑定手机、隐私设置
- **地址管理**: 添加/编辑/删除收货地址

### 4.2 我的资产
- **我的订单**: 全部订单/待付款/待收货/已完成/退款
- **我的收藏**: 收藏的商品和帖子
- **我的优惠券**: 可用/已用/已过期
- **积分中心**: 积分明细/积分兑换

### 4.3 设置
- **主题切换**: 明亮模式 ↔ 暗色模式
- **语言切换**: 中文/English/日本語/한국어/Русский
- **消息通知**: 开关各类通知推送
- **关于我们**: 版本信息/用户协议/隐私政策

## 5. 常见问题

### Q1: 忘记密码怎么办？
A: 在登录页点击 **"忘记密码"**，通过手机验证码重置。

### Q2: 如何申请退款？
A: 进入 **"我的订单"** → 点击对应订单 → **"申请退款"** → 选择退款原因。

### Q3: 配送超时怎么办？
A: 在订单详情页点击 **"联系客服"** 或拨打配送员电话。

### Q4: 如何成为入驻商家？
A: 联系平台管理员提交商家入驻申请，审核通过后开通商家后台。

### Q5: 小程序和Web端数据互通吗？
A: 是的，同一账号在Web端和小程序的数据完全同步。
