## 2. 推荐算法

### 2.1 推荐引擎

位置: `src/utils/recommendation-engine.ts`

```typescript
import recommendationEngine from '@/utils/recommendation-engine'

// 设置用户画像
recommendationEngine.setUserProfile({
  userId: '123',
  preferences: { '电子产品': 0.8, '服装': 0.6 },
  purchaseHistory: ['p1', 'p2'],
  browseHistory: ['p3', 'p4'],
  tags: ['年轻', '数码爱好者'],
})

// 设置商品信息
recommendationEngine.setProduct({
  id: 'p5',
  name: 'iPhone 15',
  category: '电子产品',
  tags: ['智能手机', '苹果'],
  price: 6999,
  rating: 4.8,
  sales: 10000,
  features: ['5G', 'A17芯片', '灵动岛'],
})
```

### 2.2 推荐方法

```typescript
// 基于内容的推荐
const contentResults = recommendationEngine.contentBasedRecommend('123', 10)

// 协同过滤推荐
const collaborativeResults = recommendationEngine.collaborativeFilteringRecommend('123', 10)

// 混合推荐（推荐）
const hybridResults = recommendationEngine.hybridRecommend('123', 10, {
  content: 0.4,
  collaborative: 0.4,
  popular: 0.2,
})

// 热门商品推荐
const popularResults = recommendationEngine.getPopularProducts(10)

// 新品推荐
const newResults = recommendationEngine.getNewProducts(10)

// 相似商品推荐
const similarResults = recommendationEngine.getSimilarProducts('p5', 10)
```

## 3. 用户画像

### 3.1 用户画像引擎

位置: `src/utils/user-profiling.ts`

```typescript
import userProfilingEngine from '@/utils/user-profiling'

// 创建用户画像
const profile = userProfilingEngine.createProfile('123', '张三')

// 更新行为指标
userProfilingEngine.updateBehaviorMetrics('123', {
  orderCount: 1,
  orderAmount: 299,
  category: '电子产品',
  device: 'mobile',
  paymentMethod: 'wechat',
})
```

### 3.2 用户标签

```typescript
// 添加标签
userProfilingEngine.addTag('123', '高消费', 0.9, 'purchase')
userProfilingEngine.addTag('123', '数码爱好者', 0.8, 'browse')

// 自动打标签
userProfilingEngine.autoTag('123')

// 获取用户标签
const tags = userProfilingEngine.getTags('123')
```

### 3.3 用户分群

```typescript
// 创建用户分群
const segment = userProfilingEngine.createSegment({
  name: '高价值用户',
  description: '消费2000以上的活跃用户',
  criteria: {
    totalSpentRange: [2000, 999999],
    orderCountRange: [10, 999],
    tags: ['活跃用户'],
  },
})

// 检查用户所属分群
const isVip = userProfilingEngine.isInSegment('123', segment.id)

// 获取用户所属分群
const userSegments = userProfilingEngine.getUserSegments('123')
```

## 4. 数据分析API

### 4.1 用户行为分析API

```typescript
import * as userBehaviorApi from '@/api/user-behavior'

// 获取用户行为数据
const behaviorData = await userBehaviorApi.getUserBehaviorData({
  userId: '123',
  startDate: '2024-01-01',
  endDate: '2024-12-31',
})
```

### 4.2 推荐追踪API

```typescript
import { recommendationTrackingApi } from '@/api/recommendation-tracking'

// 记录推荐曝光
await recommendationTrackingApi.logExposure({
  userId: 123,
  productId: 456,
  sessionId: 'session123',
  deviceType: 'mobile',
})

// 记录推荐点击
await recommendationTrackingApi.logClick({
  userId: 123,
  productId: 456,
  sessionId: 'session123',
  deviceType: 'mobile',
})

// 记录推荐转化
await recommendationTrackingApi.logPurchase({
  userId: 123,
  productId: 456,
  sessionId: 'session123',
  deviceType: 'mobile',
})

// 获取推荐效果统计
const stats = await recommendationTrackingApi.getRecommendationStatistics({
  timeRange: 'week',
})
```

## 6. 数据分析配置

### 6.1 分析维度

| 维度 | 说明 | 数据来源 |
|------|------|----------|
| 用户分析 | 用户画像、行为、价值 | 用户行为API |
| 销售分析 | 销售额、订单量、转化率 | 订单API |
| 商品分析 | 热销商品、品类分析 | 商品API |
| 流量分析 | PV、UV、访问路径 | 行为追踪API |

### 6.2 分析周期

- **日报**: 当天数据统计
- **周报**: 最近7天数据统计
- **月报**: 最近30天数据统计
- **年报**: 年度数据统计

## 8. 性能指标

### 8.1 推荐效果指标

| 指标 | 说明 | 目标 |
|------|------|------|
| CTR | 点击率 | > 5% |
| CVR | 转化率 | > 2% |
| 曝光点击比 | 曝光到点击 | > 10% |
| 点击转化比 | 点击到购买 | > 5% |

### 8.2 用户价值指标

| 指标 | 说明 |
|------|------|
| ARPU | 每用户平均收入 |
| LTV | 用户生命周期价值 |
| 留存率 | 用户留存比例 |
