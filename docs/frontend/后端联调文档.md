## 🔧 基础配置

### API基础地址

| 环境 | 地址 |
|------|------|
| 开发环境 | `http://localhost:8080/api` |
| 测试环境 | `https://test-api.heikeji.com/api` |
| 生产环境 | `https://api.heikeji.com/api` |

### 请求头配置

```http
Content-Type: application/json
Authorization: Bearer {token}
X-Request-Id: {uuid}
```

### 通用响应格式

```typescript
interface ApiResponse<T> {
  code: number        // 状态码: 0成功，其他失败
  message: string     // 提示信息
  data: T            // 响应数据
  timestamp: number   // 时间戳
  requestId: string   // 请求ID
}
```

### 通用错误码

| 错误码 | 说明 | 处理方式 |
|--------|------|----------|
| 0 | 成功 | - |
| 400 | 请求参数错误 | 检查请求参数 |
| 401 | 未授权 | 重新登录 |
| 403 | 禁止访问 | 检查权限 |
| 404 | 资源不存在 | 检查资源ID |
| 500 | 服务器错误 | 联系后端 |
| 1001 | Token过期 | 刷新Token |
| 1002 | Token无效 | 重新登录 |
