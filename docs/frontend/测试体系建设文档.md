# 测试体系建设文档

## 1. 测试基础设施

### 1.1 单元测试 (Vitest)

#### 配置
- **框架**: Vitest 4.0+
- **环境**: jsdom (DOM模拟)
- **覆盖率阈值**: 70%
- **测试超时**: 10000ms

#### 运行命令
```bash
# 运行所有单元测试
npm run test:unit

# 监听模式
npm run test:watch

# 生成覆盖率报告
npm run test:coverage

# 运行特定测试
npm run test:unit src/utils/cache.test.ts
```

#### 配置详情 (vitest.config.mjs)
```javascript
{
  test: {
    globals: true,
    environment: 'jsdom',
    include: ['src/**/*.{test,spec}.{js,mjs,cjs,ts,mts,cts,vue}'],
    coverage: {
      provider: 'v8',
      reporter: ['text', 'json', 'html', 'lcov', 'cobertura'],
      thresholds: {
        lines: 70,
        functions: 70,
        branches: 70,
        statements: 70,
      }
    }
  }
}
```

## 5. 测试最佳实践

### 5.1 单元测试规范
```typescript
// 示例：工具函数测试
import { describe, it, expect } from 'vitest'
import { formatDate } from '@/utils'

describe('formatDate', () => {
  it('should format date correctly', () => {
    const result = formatDate('2024-01-01', 'YYYY-MM-DD')
    expect(result).toBe('2024-01-01')
  })
})
```

### 5.2 组件测试规范
```typescript
// 示例：组件测试
import { mount } from '@vue/test-utils'
import { describe, it, expect } from 'vitest'
import MyComponent from './MyComponent.vue'

describe('MyComponent', () => {
  it('should render correctly', () => {
    const wrapper = mount(MyComponent, {
      props: {
        title: 'Test'
      }
    })
    expect(wrapper.text()).toContain('Test')
  })
})
```

### 5.3 E2E测试规范
```typescript
// 示例：E2E测试
import { test, expect } from '@playwright/test'

test('user login flow', async ({ page }) => {
  await page.goto('/login')
  await page.fill('[data-testid="username"]', 'testuser')
  await page.fill('[data-testid="password"]', 'password')
  await page.click('[data-testid="login-button"]')
  await expect(page).toHaveURL('/dashboard')
})
```

## 7. 故障排查

### 7.1 常见问题
1. **测试超时**: 增加 `testTimeout` 配置
2. **覆盖率不足**: 使用 `test-coverage-boost.js` 脚本提升
3. **E2E测试失败**: 检查 `playwright.config.ts` 配置

### 7.2 调试命令
```bash
# 调试模式
npm run test:e2e:debug

# 仅运行失败的测试
npm run test:unit -- --reporter=verbose --bail

# 输出详细报告
npm run test:e2e:report
```
