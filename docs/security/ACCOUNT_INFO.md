## 5. 注意事项

1. **账号安全**：
   - 定期更换密码，建议每3个月更换一次
   - 避免使用弱密码，密码应包含大小写字母、数字和特殊字符
   - 不要将账号密码泄露给无关人员

2. **权限管理**：
   - 生产环境建议创建专用账号，避免直接使用root账号
   - 根据实际需求分配最小必要权限

3. **账号备份**：
   - 定期备份账号信息，确保在紧急情况下能够恢复
   - 备份文件应妥善保管，避免泄露
