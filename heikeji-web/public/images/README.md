# 黑龙江科技大学校园图片

## 图片用途
这些图片用于首页 Hero 轮播区域的背景展示。

## 需要的图片

### 1. campus-gate.jpg
- **用途**: 首页第1张轮播 - 展示学校大门
- **建议内容**: 黑龙江科技大学校门、校名石碑、入口广场
- **尺寸建议**: 1920x1080 或更高分辨率
- **来源**: 学校官网、百度图片、实地拍摄

### 2. tangu-building.jpg
- **用途**: 首页第2张轮播 - 展示碳谷大厦
- **建议内容**: 碳谷大厦全景、建筑外观、夜晚灯光效果
- **尺寸建议**: 1920x1080 或更高分辨率
- **来源**: 学校官网新闻、建设成果展示

### 3. campus-view.jpg
- **用途**: 首页第3张轮播 - 展示校园全景
- **建议内容**: 校园鸟瞰图、图书馆、教学楼群、校园风光
- **尺寸建议**: 1920x1080 或更高分辨率
- **来源**: 学校宣传片截图、航拍图

## 图片来源建议

1. **学校官网**: http://www.usth.edu.cn
   - 新闻中心图片
   - 学校概况-校园风光
   - 招生信息网图片

2. **百度图片搜索**:
   - 搜索"黑龙江科技大学 校园"
   - 搜索"黑龙江科技大学 碳谷大厦"
   - 搜索"黑龙江科技大学 校门"

3. **学校官方新媒体**:
   - 微信公众号图片
   - 官方微博图片
   - 招生宣传册

## 图片处理建议

1. **格式**: JPG 格式，质量 80% 以上
2. **命名**: 使用英文小写，单词间用连字符
3. **大小**: 单张图片不超过 2MB
4. **版权**: 确保使用学校官方公开图片或自己拍摄的图片

## 替换步骤

1. 下载或拍摄黑龙江科技大学真实校园图片
2. 将图片重命名为:
   - `campus-gate.jpg` (校门)
   - `tangu-building.jpg` (碳谷大厦)
   - `campus-view.jpg` (校园全景)
3. 放入此目录: `heikeji-web/public/images/`
4. 刷新浏览器查看效果

## 备用方案

如果暂时无法获取真实图片，可以:
1. 使用学校官网的图片链接 (需要确保跨域允许)
2. 使用占位图片，后续替换
3. 使用 SVG 矢量图展示校园建筑轮廓
