## 通用参数设置

### Midjourney 参数
```
--ar 1:1 --v 6 --style raw
```

### DALL·E 设置
- 尺寸：1024x1024
- 风格： vivid

### 文心一格设置
- 风格：美食摄影
- 画质：高清
