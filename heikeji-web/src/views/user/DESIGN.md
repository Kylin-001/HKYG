# 用户中心页面设计文档 v2.0

## 一、设计目标

### 1.1 解决的问题
- 头像重复出现3次，信息冗余
- 页面层级混乱，用户难以快速找到功能
- 菜单过于密集，缺乏视觉呼吸感
- 交互反馈不明显

### 1.2 设计原则
- **简洁**: 去除重复元素，保留核心信息
- **层次**: 清晰的信息层级，一眼找到重点
- **反馈**: 明确的交互反馈，提升用户体验
- **品牌**: 统一的视觉风格，突出校园特色

## 三、组件详细设计

### 3.1 顶部导航栏 (Layout.vue)

#### 功能定位
- 全局导航，保持简洁
- 只保留必要元素

#### 视觉设计
```
高度: 64px
背景: 白色/透明
阴影: 滚动时出现微弱阴影
```

#### 元素组成
| 元素 | 位置 | 说明 |
|-----|------|------|
| Logo | 左侧 | 黑科易购品牌标识 |
| 搜索框 | 中间 | 可展开的全局搜索 |
| 购物车图标 | 右侧 | 显示商品数量角标 |
| 消息图标 | 右侧 | 显示未读消息角标 |
| 用户头像 | 最右侧 | 小尺寸圆形头像 |

#### 交互行为
- 滚动页面时，导航栏添加阴影 `shadow-sm`
- 头像点击跳转到个人资料页
- 图标悬停时颜色变为主题色

```

边框: 4px solid rgba(255,255,255,0.3)
圆角: 圆形 (rounded-full)
阴影: shadow-lg
在线状态: 右下角绿色圆点 (w-4 h-4, bg-pine)
悬停效果: 显示相机图标遮罩
```

#### 统计数字卡片
```
布局: 4列网格 (grid-cols-4)
间距: 12px (gap-3)
单个卡片:
  - 背景: rgba(255,255,255,0.15) + backdrop-blur
  - 边框: 1px solid rgba(255,255,255,0.2)
  - 圆角: 16px (rounded-2xl)
  - 内边距: 16px (p-4)
  - 文字居中

数字样式:
  - 字号: 24px-30px (text-2xl sm:text-3xl)
  - 字重: bold
  - 颜色: 渐变文字 (不同卡片不同颜色)
    - 发布帖子: sky-300 → cyan-200 (蓝色)
    - 收到点赞: pink-300 → rose-200 (粉色)
    - 完成订单: emerald-300 → green-200 (绿色)
    - 信用评分: violet-300 → purple-200 (紫色)

标签样式:
  - 字号: 12px (text-xs)
  - 颜色: rgba(255,255,255,0.8)
```

#### 交互行为
- **头像悬停**: 显示半透明黑色遮罩 + 相机图标
- **头像点击**: 打开文件选择器，支持上传新头像
- **统计卡片悬停**: scale(1.05) + 背景变亮 + 阴影增强
- **统计卡片点击**: 跳转到对应页面
- **设置按钮**: 右上角固定位置，悬停背景变亮

### 3.3 快捷服务网格

#### 功能定位
- 高频功能的快捷入口
- 替代原有的Tab导航

#### 视觉设计
```
背景: 白色/90%透明度 + backdrop-blur
圆角: 16px (rounded-2xl)
边框: 1px solid primary-50/50
阴影: shadow-sm
内边距: 20px (p-5)
```

#### 布局结构
```
标题: "快捷服务" + [查看更多 →]
布局: 6列网格 (grid-cols-6)
间距: 8px (gap-2)
```

#### 单个服务项
```
容器:
  - 垂直flex布局 (flex-col)
  - 居中对齐 (items-center)
  - 内边距: 12px (py-3)
  - 圆角: 12px (rounded-xl)
  - 悬停背景: primary-50/60

图标区域:
  - 尺寸: 44px × 44px (w-11 h-11)
  - 背景: 渐变 from-primary-50 to-primary-100/50
  - 圆角: 12px (rounded-xl)
  - 图标大小: 20px
  - 图标颜色: primary
  - 角标: 右上角红色圆点/数字

文字:
  - 字号: 12px (text-xs)
  - 颜色: text-secondary
  - 悬停颜色: primary
  - 上边距: 8px (mt-2)
```

#### 服务项列表
| 图标 | 名称 | 角标 | 跳转路径 |
|-----|------|-----|---------|
| Document | 全部订单 | - | /user/orders |
| Clock | 待付款 | 1 | /user/orders?status=pending_payment |
| Van | 待收货 | 1 | /user/orders?status=shipped |
| CircleCheck | 已完成 | - | /user/orders?status=completed |
| Star | 收藏夹 | 15 | /user/favorites |
| Location | 地址 | - | /user/addresses |

#### 交互行为
- **悬停**: 背景变浅 + 图标区域背景变深
- **点击**: 路由跳转到对应页面
- **角标**: 红色背景白色文字，数字大于99显示"99+"

### 3.4 最近订单区域

#### 功能定位
- 展示最近3-5个订单
- 提供快速查看入口

#### 视觉设计
```
背景: 白色/90%透明度 + backdrop-blur
圆角: 16px (rounded-2xl)
边框: 1px solid primary-50/50
阴影: shadow-sm
内边距: 20px (p-5)
```

#### 布局结构
```
标题行: "最近订单" + [查看全部 →]
内容: 横向滚动容器
  - 显示3-5个订单卡片
  - 超出部分可左右滑动
  - 隐藏滚动条但保持可滚动
```

#### 订单卡片
```
尺寸: 固定宽度 200px
背景: gray-50
圆角: 12px
内边距: 12px

内容:
  - 商品图片 (左侧, 60px × 60px)
  - 商品名称 (单行省略)
  - 订单状态标签
  - 订单金额
```

#### 空状态
```
显示: "暂无订单" 提示
按钮: [去逛逛 →] 跳转到首页
```

#### 交互行为
- **横向滑动**: 触摸设备支持左右滑动
- **卡片点击**: 跳转到订单详情
- **查看全部**: 跳转到订单列表页
