## ⚠️ 遇到的问题

### 1. Vitest配置弃用警告
**问题**: 
```
DEPRECATED `test.poolOptions` was removed in Vitest 4. All previous `poolOptions` are now top-level options.
```

**解决方案**: 将poolOptions移到顶层配置

### 2. baseline-browser-mapping过期警告
**问题**:
```
The data in this module is over two months old.
```

**影响**: 不影响测试执行，仅是警告

**解决方案**: 后续可以更新依赖

### 3. ESM模块兼容性问题
**问题**: jsdom与某些ESM模块存在兼容性问题

**解决方案**: 
- 配置单进程模式（singleFork: true）
- 已启动依赖更新
