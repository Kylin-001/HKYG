| 文档名称 | 描述 | 适用场景 |

## 2. 文档使用指南

### 2.1 新用户快速入门

1. **第一步**：阅读 `README.md` 了解项目概述
2. **第二步**：参考 `DEPLOYMENT_GUIDE.md` 进行部署
3. **第三步**：根据需要参考相关开发文档

### 2.2 部署场景指南

| 场景 | 推荐文档 |
|------|----------|
| 开发环境部署 | DEPLOYMENT_SIMPLIFIED.md |
| 测试环境部署 | DEPLOYMENT_GUIDE.md |
| 生产环境部署 | DEPLOYMENT_OPTIMIZATION.md |
| 部署问题排查 | DEPLOYMENT_SOLUTIONS.md |
| MySQL问题 | 对应的MYSQL_*.md文档 |

### 2.3 文档更新和维护

1. **新增文档**：将新文档放入对应的分类目录
2. **更新文档**：确保相关文档保持一致
3. **删除文档**：删除重复或过时的文档
4. **版本控制**：使用Git进行文档版本管理
