## 🏗️ 项目架构

### 微服务架构 (10个服务)

| 服务 | 端口 | 说明 |
|------|------|------|
| API网关 | 8080 | 请求路由、权限验证 |
| 管理后台 | 8090 | 后台管理系统 |
| 用户服务 | 8081 | 用户管理、认证 |
| 商品服务 | 8082 | 商品管理 |
| 订单服务 | 8083 | 订单管理 |
| 配送服务 | 8001 | 配送管理 |
| 会员服务 | 8088 | 会员权益 |
| 校园服务 | 8003 | 校园服务 |
| 二手服务 | 8006 | 二手交易 |
| 失物招领 | 8007 | 失物招领 |

### 技术栈

- **前端**: Vue 3 + Element Plus
- **后端**: Spring Cloud Alibaba
- **注册/配置中心**: Nacos
- **缓存**: Redis
- **消息队列**: RabbitMQ
- **数据库**: MySQL
