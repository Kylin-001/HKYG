# 黑龙江科技大学数据导入指南

## 项目概述

本指南提供了如何将黑龙江科技大学相关数据导入到黑科技商城项目数据库的详细步骤。

## 准备工作

1. **确保数据库已创建**：
   - 数据库名称：`heikeji-mall`
   - 字符集：`utf8mb4`
   - 排序规则：`utf8mb4_unicode_ci`

2. **检查数据库连接**：
   - 主机地址：`192.168.110.162`
   - 端口：`3306`
   - 用户名：`root`
   - 密码：请使用您的MySQL root密码

3. **确保MySQL服务正在运行**：
   ```bash
   # Windows
   net start mysql
   
   # Linux
   sudo systemctl start mysql
   ```

## 数据文件说明

我们已经创建了包含黑龙江科技大学相关数据的SQL文件：

- **文件路径**：`e:\Program File\黑科易购项目\heikeji-mall\sql\heikeji_data.sql`
- **包含数据**：用户信息、商家信息、商品分类、商品数据、收货地址、校园信息、外卖柜、跑腿员、购物车、订单等

## 导入步骤

### 方法一：使用MySQL命令行工具

1. **打开命令行窗口**（Windows：CMD或PowerShell，Linux：终端）

2. **切换到项目根目录**：
   ```bash
   cd "e:\Program File\黑科易购项目\heikeji-mall"
   ```

3. **执行导入命令**：
   ```bash
   # Windows
   mysql -h 192.168.110.162 -P 3306 -u root -p heikeji-mall < "sql\heikeji_data.sql"
   
   # Linux
   mysql -h 192.168.110.162 -P 3306 -u root -p heikeji-mall < sql/heikeji_data.sql
   ```

4. **输入MySQL密码**：执行命令后，系统会提示您输入MySQL root用户的密码

### 方法二：使用图形化工具（如MySQL Workbench）

1. **打开MySQL Workbench**并连接到数据库服务器

2. **选择数据库**：在左侧导航栏中选择`heikeji-mall`数据库

3. **执行SQL文件**：
   - 点击菜单栏的`File` → `Open SQL Script`
   - 选择`e:\Program File\黑科易购项目\heikeji-mall\sql\heikeji_data.sql`文件
   - 点击`Execute`按钮（或按`Ctrl+Shift+Enter`）执行脚本

### 方法三：使用Navicat

1. **打开Navicat**并连接到数据库服务器

2. **选择数据库**：在左侧导航栏中展开连接，选择`heikeji-mall`数据库

3. **执行SQL文件**：
   - 右键点击`heikeji-mall`数据库
   - 选择`Run SQL File`
   - 选择`e:\Program File\黑科易购项目\heikeji-mall\sql\heikeji_data.sql`文件
   - 点击`Start`按钮执行脚本

## 常见问题解决

1. **连接失败**：
   - 检查MySQL服务器是否正在运行
   - 检查网络连接
   - 确保MySQL允许远程连接（修改`my.cnf`或`my.ini`文件中的`bind-address`配置）
   - 检查防火墙设置，确保3306端口已开放

2. **权限问题**：
   ```sql
   -- 授予root用户远程访问权限
   GRANT ALL PRIVILEGES ON *.* TO 'root'@'%' IDENTIFIED BY 'your_password' WITH GRANT OPTION;
   FLUSH PRIVILEGES;
   ```

3. **导入失败**：
   - 检查SQL文件路径是否正确
   - 确保数据库已存在且字符集正确
   - 查看导入错误信息，针对性解决
